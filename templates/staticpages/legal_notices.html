    Informations techniques

    Le code source du site est écrit en Python 3 et utilise le framework Django.

    Le code source du site est disponible sur le GitHub du TamiaLab, sous licence AGPLv3.

    Le stack logiciel du site est basé sur un (classique) trio Nginx, Gunicorn, Supervisord, avec une pinçée de Memcached pour le cache côté serveur et un nuage de Postgresql pour les bases de données. Le moteur de recherche du site est basé sur HayStack, avec ElasticSearch comme moteur d'indéxation. Le tout vous est servi sur un plateau par un serveur GNU/Debian, customizé à la sauce TamiaLab.

    Les noms de domaine sont fournis par OVH. Le certificat SSL servant à sécuriser vos communications avec le site est fourni par Gandi.net.

    Le développement du site est réalisé au moyen du fabuleux IDE PyCharm Pro, avec l'aide de Virtual Box comme gestionnaire de virtualisation et de Vagrant comme gestionnaire d'environnements de développement virtualisés.
